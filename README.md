## JPS - JavaScript Plugin System

### About

This plugin system for JS was created in *40 minutes*,
thinking about making the library system easier and simpler to do than NPM

### How to use

**1**
Download version .rar template and extract it

**2**
Customize and make your plugin, or use a ready-made plugin

**3**
Open the folder and the terminal inside folder

**4**
Write 'node exec/worker.js'
[OPTIONAL] 'cd to exec/ folder, with it, you can use only 'node worker.js'

**5**
Put your project folder path
Example: *C:\Users\ksdev\Downloads\JPS Test*

Now, the plugin will be installed inside *C:\Users\ksdev\Downloads\JPS Test\js-plugins\template-plugin\*
